# Interviews

Drop each new interview in `inbox/<slug>/` (e.g. `inbox/jane-doe/`):

- `transcript.md` / `.txt` / `.vtt` — required
- `notes.md` — name, current role, degree(s), preferred link (recommended)
- headshot `.jpg`/`.png` — optional

Then tell Claude "new issue" — the `new-issue` skill drafts the story page, refreshes the opportunities board, and drafts the newsletter issue. Processed interviews move to `processed/`.
